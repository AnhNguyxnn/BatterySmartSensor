Battery Monitor Firmware Upload

Upload firmware từ máy local lên backend server

Hướng dẫn sử dụng:

  1. Build firmware trên máy local với PlatformIO
  2. Chọn file firmware .bin đã build
  3. Nhập thông tin version và server
  4. Click "Upload Firmware"
  5. ESP32 sẽ tự động phát hiện firmware mới

Alternative Methods:

1. Command Line (Bash):

./upload_firmware_local.sh firmware.bin 1.0.1 http://cloud.anhnguyxn.io.vn:8000

2. Python Script:

python3 upload_firmware.py firmware.bin 1.0.1 http://cloud.anhnguyxn.io.vn:8000

3. curl Command:

curl -X POST http://cloud.anhnguyxn.io.vn:8000/api/firmware/upload \ -H "X-API-Key: battery_monitor_2024_secure_key" \ -F "file=@firmware.bin" \ -F "version=1.0.1" \ -F "build=1" \ -F "release_notes=Bug fixes"
